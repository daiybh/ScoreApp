需要一个 wetChat小程序，和一个web服务端（python）

# wetchat 功能
## 输入成绩 

1. 当前成绩，
2. 班级最高分，
3. 班级平均分，
4. 班级排名
5. 科目
6. 考试内容
7. 考试时间

## 输出成绩
 
根据科目列举出所有成绩


# web服务端功能

接受输入，并把数据存储到7牛云上，
数据按照 姓名\科目\考试时间.json 的格式保存

 ## 接受输入

根据 参数 姓名/科目/考试时间 创建对应的json 文件 并保存到7牛云上

## 输出
2. 输出成绩

根据 参数 姓名/科目  遍历所有对应json ，并组织成完整json 输出
