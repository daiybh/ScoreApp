# 成绩管理系统 - 微信小程序

## 功能介绍

本微信小程序是成绩管理系统的客户端，提供以下功能：

1. **添加成绩**：输入学生的成绩信息并提交到服务器
2. **查看成绩**：根据姓名和科目查询学生的成绩记录

## 使用说明

### 添加成绩

1. 进入"添加成绩"页面
2. 填写以下信息：
   - 姓名（必填）
   - 当前成绩（必填）
   - 班级最高分（选填）
   - 班级平均分（选填）
   - 班级排名（选填）
   - 科目（必填）
   - 考试内容（选填）
   - 考试时间（必填）
3. 点击"提交"按钮保存成绩

### 查看成绩

1. 进入"查看成绩"页面
2. 输入要查询的学生姓名（必填）
3. 可选择输入科目进行筛选
4. 点击"查询"按钮获取成绩记录

## 技术架构

- 使用微信小程序原生开发框架
- 通过HTTP请求与后端API进行数据交互
- 响应式设计，适配不同尺寸的移动设备

## 配置说明

在 `app.js` 文件中配置服务器地址：

```javascript
globalData: {
  serverUrl: 'http://localhost:5000/api'  // 修改为实际的服务器地址
}
```

## 注意事项

1. 确保网络连接正常，能够访问服务器
2. 提交成绩时，必填项不能为空
3. 考试时间格式为 YYYY-MM-DD
